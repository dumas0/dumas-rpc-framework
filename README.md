# dumas-rpc-framework
Simple RPC project for learning use only.

## 目录结构
```
dumas-rpc-framework
├─config                  -- 配置
├─rpc-client              -- rpc 客户端
├─rpc-framework-common    -- rpc 框架通用模型
├─rpc-framework-simple    -- 简单 rpc 框架
├─rpc-server              -- rpc 服务端
├─rpc-service-api         -- rpc 接口
└─docs                    -- 文档模块
```

## 开发环境
- [x] Java 开发工具包 1.8+
- [x] IDE（Eclipse或IntelliJ IDEA）
- [x] 项目管理工具 Maven
- [x] 注册中心 Zookeeper
